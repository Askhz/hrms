# hrms
基于go、gorm、gin、mysql及layui构建的人力资源管理系统。提供员工管理、考试管理、薪资考勤管理、权限管理及分公司分库数据隔离等功能。欢迎Star或提Issue。

# 使用方式
- git clone https://github.com/pbrong/hrms.git
- cd hrms && go mod tidy
- 更新conf配置文件配置
- sh build.sh 执行脚本编译可执行文件执行 或 直接启动main.go运行
